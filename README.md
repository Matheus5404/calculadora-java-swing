Aplicação desenvolvida em Java utilizando Java Swing, com o objetivo de praticar conceitos de Programação Orientada a Objetos, interface gráfica e lógica matemática.

O sistema permite realizar operações básicas de cálculo de forma simples e intuitiva.

🚀 Demonstração do Sistema

A animação acima mostra a calculadora sendo utilizada em tempo real.

🎯 Funcionalidades

✅ Soma
✅ Subtração
✅ Multiplicação
✅ Divisão
✅ Limpar visor
✅ Interface gráfica amigável

🛠️ Tecnologias utilizadas

Java

Java Swing

Programação Orientada a Objetos (POO)

IDE NetBeans
